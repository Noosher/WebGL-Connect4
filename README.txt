WebGL Connect 4:
This is a simple version of connect 4 that you can play directly with a friend in your web browser using WebGL.

Team members:
Jackson Sulik and Robert Urton

How to set up:
1. Unzip the file the project is in
2. Open the game by right clicking main.html, open with a web browser

How to Play:
1. Use the left and right arrow keys to decide which column to use.
2. Press the down arrow to drop the circle to the lowest row.
3. When someone wins press restart and go back to step 1.

Tools Used:
HTML, basic levels of CSS, JavaScript, WebGL
